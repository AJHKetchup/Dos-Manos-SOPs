## How to use these drafts

1. Use the SOP that matches the work surface.
2. Keep customer, finance, HR, legal, claim, payment, and restricted details inside approved systems or restricted local caches.
3. Treat live mutations as approval-gated unless the SOP explicitly says the action is safe and reversible.
4. If a source document conflicts with these drafts, follow the more specific source document and update this folder after review.
5. If a nontechnical human needs a Drive or PDF copy, export from this repo and clearly mark the export as a derivative.
