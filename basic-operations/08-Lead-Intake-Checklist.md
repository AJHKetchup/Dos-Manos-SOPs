# Lead Intake Checklist

Status: Draft helper checklist
Last updated: 2026-07-03

Use with `03-Lead-Intake-and-Quote-Handoff-SOP.md`.

## Quick checklist

- [ ] Customer name
- [ ] Phone
- [ ] Email
- [ ] Best callback window
- [ ] Move date or target window
- [ ] Pickup address / ZIP
- [ ] Delivery address / ZIP, if known
- [ ] Property type at pickup
- [ ] Property type at delivery
- [ ] Stairs, elevators, long carries, loading dock, parking, or building rules
- [ ] Inventory or room-by-room/item list
- [ ] Heavy, fragile, sentimental, high-value, awkward, or difficult items
- [ ] Packing/unpacking needs
- [ ] Added stops
- [ ] COI/building requirements
- [ ] How the customer heard about Dos Manos
- [ ] Referral partner/person, if any
- [ ] Lead status
- [ ] Next follow-up date
- [ ] Escalation issue, if any

## Escalate before quoting or promising

- Pricing exception or discount requested.
- Incomplete/vague inventory.
- Interstate move.
- Specialty/heavy/high-risk item.
- COI/additional insured request.
- Claim, complaint, damage, refund, legal, ICC, or chargeback issue.
- Public-review threat.
- Unusual building rule or tight deadline.
- Any service claim that is not already approved.
